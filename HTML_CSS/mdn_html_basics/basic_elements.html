< p > in here is a tag. The below text is enclosed in paragraph tags:

My cat is very grumpy

Attributes can contain extra information about the element, but these don't appear in the page content:

My cat is very grumpy

You can nest elements:

My cat is very grumpy

Some elements have no content and are called void elements. The img element is one of those. Also note there is no closing img tag and no inner content.
