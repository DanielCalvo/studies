# Section 23: Machine Learning

### 252. Rekognition Overview
Rekognition is a service used to find objects, people, text, and scenes in images and videos using machine learning
So you can use it for things like labeling, content moderation, text detection, face detection, and analysis for things like emotions, and recognize certain people

Apparently it's really useful for content moderation

### 253. Transcribe Overview
Automatically converts speech to text

So it's useful for things like transcribing customer service calls, automating closed captioning and subtitling, and generating metadata for media assets to create a fully searchable archive

### 254. Polly Overview
Polly is the opposite of Transcribe, you turn text into lifelike speech using deep learning

You can add pronunciation lexicons so it turns things like acronyms into full-sized words like with "AWS"

### 255. Translate Overview
Allows you to localize content and translate stuff

### 256. Lex + Connect Overview
Allows you to build a chatbot for a call center, similar to what Alexa does

### 257. Comprehend Overview
For natural language processing, fully managed and serverless, can extract context from the text, do sentiment analysis, and so on
So for instance, a use case is to go over customer support emails and see if the interactions were positive or negative

### 258. Comprehend Medical Overview
Returns useful information from unstructured clinical text

### 259. SageMaker AI Overview
A fully managed service for developers and data scientists to build machine learning models

### 260. Kendra Overview
A fully managed document search service powered by machine learning, and this extracts answers from within a document

### 261. Personalize Overview
A machine learning service to build applications with real-time personalized recommendations

### 262. Textract Overview
Used to extract text, handwriting, and data from any scanned documents using AI and machine learning
